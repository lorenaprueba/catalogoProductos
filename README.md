# Catálogo de Productos

Este es el proyecto base para la aplicación móvil Android del catálogo de productos.

## Tecnologías Utilizadas
- **Lenguaje:** Kotlin
- **Navegación:** Android Navigation Component 
- **Estructura y UI:** Material Design
- **Herramienta:** Android Studio Iguana/Jellyfish+

## Estructura de Secciones
El aplicativo incluye una navegación estática lateral usando `NavigationRailView` hacia los siguientes fragmentos:
- Inicio
- Catálogo
- Favoritos
- Carrito
- Perfil
- Web

## Instalación
Para compilar y ejecutar el proyecto:
1. Clonar el repositorio.
2. Abrir en **Android Studio**.
3. Dejar que **Gradle** sincronice las dependencias del `libs.versions.toml`.
4. Ejecutar usando un emulador o un dispositivo físico Android.
